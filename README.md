# MemScope

MemScope 是一个用于学习 C 和 Python 内存模型的可视化演示工具。它通过变量盒子、对象盒子、列表单元格和箭头关系，把变量赋值、列表结构、引用、浅拷贝和深拷贝等概念展示出来，帮助初学者更直观地理解程序运行时的数据关系。该工具是作者作为计算机学生的作业，内容不够完整与严谨，仅供参考。

## 项目特点

- 使用 HTML、CSS 和 JavaScript 编写，无需安装额外依赖。
- 支持 Python 变量赋值和 C 变量赋值的对比展示。
- 支持一维列表、二维列表、引用、浅拷贝和深拷贝的可视化演示。
- MemScope 2.0 增加了自定义变量值和分步演示功能。
- 所有功能都可以直接在浏览器中运行。

## 功能介绍

### Python 变量赋值

展示 Python 中变量名与对象之间的引用关系，说明变量并不是直接保存值本身，而是指向对象。

### C 变量赋值

展示 C 语言中变量通常直接保存值，并通过不同地址说明 `int a` 和 `int b` 是两个独立的内存空间。

### 列表展示

用可视化单元格展示一维列表和二维列表的结构，帮助理解列表下标和嵌套列表。

### 引用、浅拷贝和深拷贝

通过图形化方式展示：

- 引用：多个变量指向同一个对象。
- 浅拷贝：复制外层对象，但内部对象仍然共享。
- 深拷贝：递归复制外层和内部对象，两个对象互不影响。
